# agate

Этот проект представляет собой адаптивную верстку главной страницы для ресторана. Страница создана с использованием HTML, CSS и современных подходов к адаптивной верстке, обеспечивая корректное отображение на разных устройствах: от мобильных телефонов до десктопов.

***Особенности:***

Адаптивный дизайн (media queries)

Современная семантическая разметка HTML5

CSS Flexbox для компоновки блоков

Чистый и минималистичный интерфейс
